Index of Supplementary Materials

Title of paper: Digraph Arborescences and Matrix Determinants

Authors: Sayani Ghosh and Bradley S. Meyer

File: Examples.pdf

Type: Pdf file

Contents: A pdf file giving examples illustrating the content of the paper.

File: compute_determinant.py

Type: Python code

Contents: A Python code to compute determinants from the sum of arborescence weights and from LU decomposition. The routine can also print out pdf figures of the arborescences. The code is maintained at this GitHub website. That site also gives instructions for the code's use.

File: create_random_matrix.py

Type: Python code

Contents: A Python code to create a random matrix of arbitrary size for testing the branching coees. The code is maintained at this GitHub website. That site also gives instructions for the code's use.

File: factor_determinant.py

Type: Python code

Contents: A Python routine to factor matrix determinants graphically and compute the determinant from the sum of arborescence weights and from LU decomposition. The routine can also print out pdf figures of the factored graphs. The code is maintained at this GitHub website.

File: matrix_graph.py

Type: Python code

Contents: A Python helper code to create matrix digraphs from input matrices and to compute matrix determinants from LU decomposition. The code is maintained at this GitHub website.

File: rootify.py

Type: Python code

Contents: A Python helper code to perform vertex isolation in matrix digraphs. The code is maintained at this GitHub website.
